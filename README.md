# Anotações
## Paramentros
- Routes Params
    - Ex: `http://localhost:3000/produtos/:id ou {id}`

- Query Params
    - Ex: `http://localhost:3000/produtos?name=keyboard&&color=red `
- Body Params
    - Ex: 
        ```js  
        {
            "produto": "teclado",
            "cor": "vermelho"
        }
        ```
